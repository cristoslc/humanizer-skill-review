# Musings

Pre-artifact thought capture. Freeform markdown — raw ideas, half-formed concepts, exploratory notes, and fragments that may later coalesce into ADRs, specs, or plans.

Files here follow no fixed format. The directory name signals content maturity: musings are less structured than plans, which are less structured than ADRs.

## Rules

- Commit and push immediately after writing a musing.
- See the Muse ritual (`~/.agents/agents-md-detail/locations/musings.md`) for the full protocol and deletion policy.
