# Tech Debt

Pre-existing LSP errors, lint violations, type errors, and other tech debt too complex to fix inline.

This is a documentation-only repo with no code, so tech debt entries are not expected. If shell scripts grow complex enough to warrant type-checking or linting, add entries here.

See `~/.agents/agents-md-detail/lsp-errors.md` for entry format, deletion policy, and the running index convention.
